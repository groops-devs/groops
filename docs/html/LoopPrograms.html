LoopPrograms

This program runs a list of programs in a loop.

If continueAfterError=yes and an error occurs, the remaining programs in the current iteration are skipped and the loop continues with the next iteration. Otherwise an exception is thrown.

If this program is executed on multpile processing nodes, the iterations can be computed in parallel, see parallelization. The first process serves as load balancer and the other processes are assigned to iterations according to processCountPerIteration. For example, running a loop containing three iterations on 13 processes with processCountPerIteration=4, runs the three iterations in parallel, with each iteration being assigned four processes. With parallelLog=yes all processes write output to screen and the log file.
